## 🚀 本地运行

直接双击 `index.html` 用浏览器打开即可，无需任何依赖！

## ☁️ 部署到 Cloudflare Pages / GitHub Pages

1. 把整个 `openclaw123/` 目录推送到 GitHub 仓库
2. 在 Cloudflare Pages 或 GitHub Pages 中选择该目录作为根目录
3. 构建命令留空，输出目录设为 `/`（或 `openclaw123/`）
4. 完成！纯静态网页，零成本托管
