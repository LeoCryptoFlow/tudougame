# 极简蛋仔派对 (Egg Party Potato)

一个使用HTML5 Canvas开发的网页小游戏合集。

## 游戏列表

### 🥔 极简蛋仔派对
**文件:** `egg_party_potato.html`

一个有趣的跑酷类游戏！控制一个可爱的土豆角色，躲避从右侧飞来的方块障碍物。

**特点:**
- 简单的物理引擎实现
- 流畅的跳跃和翻滚动作
- 实时分数计算
- 动态难度提升

**操作方法:**
- 点击屏幕或按空格键：跳跃
- 双击或连续按空格：二段跳
- 躲避障碍物，生存时间越长分数越高！

### 其他游戏
- `paw_patrol_galaxy.html` - 汪汪队银河冒险
- `sunwukong_galaxy.html` - 孙悟空银河冒险
- `potato_vs_zhubajie.html` - 土豆大战猪八戒

## 如何游玩

1. 下载或克隆此仓库
2. 用浏览器打开任意HTML文件即可开始游戏
3. 无需安装任何依赖，开箱即用！

## 技术栈

- HTML5 Canvas
- 原生JavaScript
- CSS3

## 在线游玩

下载后直接在浏览器中打开HTML文件即可，或者使用以下方式：

```bash
# 克隆仓库
git clone https://github.com/你的用户名/shengcode.git

# 进入目录
cd shengcode

# 用浏览器打开游戏
open egg_party_potato.html  # macOS
# 或者
start egg_party_potato.html  # Windows
```

## 开发

所有游戏都是单文件HTML，包含了完整的游戏逻辑、样式和资源。可以直接编辑HTML文件来修改游戏。
