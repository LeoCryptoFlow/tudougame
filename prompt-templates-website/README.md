# AI提示词模板网站

一个收集和分享高质量AI提示词模板的网站。

## 功能特点

- 📚 精选提示词模板库
- 🔍 强大的搜索功能
- 🏷️ 分类浏览（写作、编程、商业等）
- 📋 一键复制功能
- 🎨 响应式设计，适配各种设备

## 技术栈

- HTML5
- CSS3
- JavaScript ES6+

## 快速开始

1. 克隆项目到本地
2. 在浏览器中打开 `index.html`
3. 开始浏览和使用提示词模板！

## 项目结构

```
prompt-templates-website/
├── index.html       # 主页面
├── styles.css       # 样式文件
├── script.js        # 交互逻辑
└── README.md        # 项目说明
```

## 贡献

欢迎提交新的提示词模板！只需在 `script.js` 中的 `promptTemplates` 数组中添加新的模板对象即可。
